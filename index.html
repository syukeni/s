プリセット

映像の質感

重いと感じたら、 発光 と やわらかさ を少し下げると軽くなります。 人物には「柔らかな魂」、風景には「神秘の光」や「緑の生命」がおすすめです。

像／影の色

黒・緑・桃色・白の方向へ明暗を移し、 形と影が絵画的な色面として見えるようにします。

輝きの色

青・赤・黄の流れで、影から光へ移る発色を強調します。 発光感と相性がよいモードです。

仕上げ効果

Bloom、Soft Focus、Vignette、Grain により、 ただの色変換ではなく作品らしい空気感を加えます。

GitHub Pages に反映したら、公開URLを Safari で開いて使ってください。 保存画像には現在の見た目がそのまま書き出されます。
